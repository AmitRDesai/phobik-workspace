# Love Landing — User Scenarios

## Scenario 1: Discovering the Love Section from Practices

A user explores the Practices tab and finds the Love category.

1. User taps the **Love** card on the Practices screen
2. **Love Landing** screen opens showing:
   - A back button in the top left
   - Large "LOVE" title
   - Subtitle: "Deepen self-compassion and cultivate heart-centered awareness."
   - Two large editorial cards with background images
3. User scrolls down to see both cards

---

## Scenario 2: Accessing Love from the Today Screen

A user uses the quick access grid on their home screen.

1. User is on the **Today** screen
2. Sees four quick access buttons: Library, Courage, Love, Journal
3. Taps the **Love** button (heart icon)
4. **Love Landing** screen opens with the same content as Scenario 1

---

## Scenario 3: Starting the Empathy Challenge

A user chooses the 7-Day Empathy Challenge from the Love landing.

1. User sees the first card with a warm atmospheric image
2. Card shows:
   - Heart icon in a circular container
   - Title: "7-Day Empathy Challenge"
   - Description: "A week to strengthen awareness, connection, and compassion."
   - "Start Journey" button with an arrow icon
3. User taps the card or the "Start Journey" button
4. Navigates to the **Empathy Challenge** flow

---

## Scenario 4: Starting the Gentle Letter Practice

A user chooses the Gentle Letter practice from the Love landing.

1. User scrolls to the second card with an abstract golden swirl image
2. Card shows:
   - Edit icon in a circular container
   - Title: "Gentle Letter to Yourself"
   - Description: "A Phobik practice in courage and kindness."
   - "Begin Practice" button with a sparkle icon
3. User taps the card or the "Begin Practice" button
4. Navigates to the **Gentle Letter** intro screen

---

## Scenario 5: Navigating Back

A user decides to go back to the previous screen.

1. User taps the back arrow button in the top left
2. Returns to whichever screen they came from (Practices, Today, or Courage Options)

---

## Edge Cases

- **Slow network**: Background images on the cards may take a moment to load; the gradient overlay ensures text remains readable while images load
- **Small screen**: Cards stack vertically and are scrollable; content adjusts to fit narrower devices
