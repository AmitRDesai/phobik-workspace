# The Pivot Point Assessment — User Scenarios

## Scenario 1: Starting a New Assessment

1. User taps **Start Test** on **The Pivot Point** card in the Assessment Hub
2. The **Intro** screen appears with a badge reading "Assessment"
3. The title "The Pivot Point" is shown with the subtitle "How do you respond when life gets hard?"
4. A description explains: "When life gets stressful, we don't rise to our intentions — we fall back on patterns"
5. Two info cards show:
   - **Duration**: 8–10 minutes
   - **Insights**: Personalized Pattern
6. A privacy note reads "Your responses are private and encrypted"
7. User taps **Start Assessment**
8. The first question appears

## Scenario 2: Answering Questions

1. The question screen shows progress: "Question 1 of 50" with a percentage and animated gradient bar
2. A section label appears (e.g., "Section 1: Your First Reaction" with subtitle "When something stressful or uncertain happens…")
3. The question is displayed in bold (e.g., "I immediately try to fix or control the situation")
4. Five rating circles (1–5) appear with "Not like me" under 1 and "Very much like me" under 5
5. User taps a number — the circle fills with gradient and a label appears below (e.g., "3 = Sometimes like me")
6. A context card at the bottom shows an inspirational quote about the Pivot Point
7. User taps **Next Question** — the progress bar animates and the screen scrolls to the top

**Key point:** The 50 questions are grouped into 5 sections of 10. The section label changes as the user progresses through the assessment.

## Scenario 3: Navigating Between Questions

1. User is on question 15 and wants to revisit question 14
2. User taps the back arrow in the header
3. Question 14 appears with the previously selected rating still highlighted
4. User can change their answer and tap **Next Question** to continue

## Scenario 4: Quitting and Resuming

1. User is on question 23 and needs to take a break
2. User taps the back arrow on question 1 (or navigates away)
3. A dialog asks "Quit Assessment?" with "Your progress will be saved"
4. User taps **Quit** and returns to the Assessment Hub
5. Later, the user returns and sees a yellow **Resume** button on The Pivot Point card
6. User taps **Resume**, then **Resume Assessment** on the intro screen
7. The assessment loads at question 23 with all 22 previous answers intact

## Scenario 5: Completing the Assessment

1. User reaches question 50 — the button reads **See Results**
2. User selects a rating and taps **See Results**
3. The **Results** screen appears with the heading "This is how you respond under pressure"
4. The subheading reads "This isn't who you are — it's a pattern your brain has learned. And patterns can change."
5. Results are displayed in a card layout:
   - **Primary Pattern** — The dominant stress response (e.g., "The Pusher 🔥") with strength and growth edge
   - **Regulation Score** — A circular progress ring showing the percentage (e.g., 75%)
   - **Secondary Pattern** — The second most common response (e.g., "The Pleaser 🤝")
6. A closing message encourages: "Your next step isn't to become someone new… It's to respond just 1% differently when it matters"
7. User taps **Back to Assessments** to return to the hub

## Scenario 6: Understanding the Results

The assessment identifies one of five stress response patterns:

- **The Pusher** 🔥 — Handles stress by doing more. Strength: driven, proactive. Growth edge: slowing down, feeling more
- **The Escaper** 🏃 — Handles stress by avoiding it. Strength: protects energy. Growth edge: taking small action
- **The Freezer** 🧊 — Handles stress by shutting down. Strength: sensitive, aware. Growth edge: safe activation
- **The Pleaser** 🤝 — Handles stress by prioritizing others. Strength: empathetic, relational. Growth edge: boundaries + self-trust
- **The Regulator** 🌊 — Can feel stress and still move forward. Strength: balanced, resilient. Growth edge: continued expansion

The **Regulation Score** reflects how strongly the user aligns with the Regulator pattern — a higher score indicates greater emotional regulation capacity.

## Edge Cases

### Section Transitions
- When the user moves from question 10 to 11, the section label changes from "Your First Reaction" to "Thought Patterns" with a new subtitle
- There is no interstitial screen between sections — the transition is seamless

### Incomplete Assessment on Results Screen
- If a user somehow reaches the results screen without answering all 50 questions, a message reads "Assessment incomplete" with a button to return to the hub
