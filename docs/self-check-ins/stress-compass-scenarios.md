# The Stress Compass — User Scenarios

## Scenario 1: Starting the Assessment

1. User taps **Start Test** on **The Stress Compass** card in the Assessment Hub
2. The **Stress Compass** screen appears with the header "Stress Compass — Assessment"
3. An introduction explains what the Stress Compass is and how it maps nervous system patterns
4. A "How it works" card explains the 1–10 rating scale: 1 means "draining me" and 10 means "feels balanced"
5. Below are 10 stressor category cards, each with a slider

## Scenario 2: Rating Stressors

1. Each card shows a category name and icon (e.g., "Work & Performance" with a trending-up icon)
2. A gradient slider track runs from left to right with a draggable thumb
3. The user drags the thumb to set their rating — labels show "Draining" (left) and "Balanced" (right)
4. The current rating number is displayed (e.g., "Rating: 7")
5. User rates all 10 categories at their own pace:
   - Work & Performance
   - Financial Security
   - Relationships & Conflict
   - Self Image and Comparison
   - Time Scarcity
   - Inner Critic
   - Isolation and Loneliness
   - Unresolved Fear
   - Lack of Purpose
   - Exhaustion and Recovery
6. User taps **Generate My Map** at the bottom

## Scenario 3: Viewing the Stress Signature Map

1. The **Stress Signature Map** screen appears with an orbital visualization
2. The visualization shows three concentric rings:
   - **Inner ring** (cyan) — Balanced stressors (rated 7–10)
   - **Middle ring** (gray) — Moderate stressors (rated 4–6)
   - **Outer ring** (pink) — Drained stressors (rated 1–3)
3. Each stressor appears as a circle with its emoji positioned on the appropriate ring
4. A legend at the bottom shows "Drained" (pink) and "Balanced" (cyan)
5. Below the visualization, the user's **Top 3 Stressors** are listed as detailed cards

## Scenario 4: Understanding Top Stressors

1. Each of the three stressor cards shows:
   - The stressor name and emoji (e.g., "Financial Security 💰")
   - A priority label (e.g., "Priority Focus", "Social Bond", "Cognitive Load")
   - A **Biological Root** section explaining why this stressor affects the body
   - A **Phobik Practice** section recommending a specific exercise to help
2. User can scroll through all three cards
3. User taps **Save to Insights** to complete the assessment and return home

## Scenario 5: Adjusting Ratings Before Generating

1. User has rated all 10 stressors but wants to change one before generating the map
2. User scrolls back up to the relevant card and adjusts the slider
3. The rating number updates immediately
4. User scrolls back down and taps **Generate My Map**

## Edge Cases

### Default Ratings
- All sliders start with pre-set default values spread across the range so the user sees an example of the scale before making changes
- The defaults are not saved until the user explicitly generates the map

### Returning After Completion
- After tapping "Save to Insights," the user is taken home and the stressor ratings are reset to defaults
- Starting the Stress Compass again begins a fresh assessment
